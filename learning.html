Overview

This section will teach you what the error codes are, as well as the characteristics of each one. You will need to find these codes in the game to progress. A reference to MDN Web Docs is provided for deeper learning.

The Beginning of Something Big

Begin your learning journey today by reading a one-sentence summary of each 4XX HTTP status code (client-side). There are, however, certain codes omitted from the game, as I am unable to replicate them.
